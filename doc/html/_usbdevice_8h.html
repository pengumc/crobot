int Usbdevice_connect ( usbdevice_t *  usbdevice)

Connect the device.

Prints to stderr if the device could not be connected.
The connected variable is lowered until a succesfull connect On success, the connected variable is set to USBDEV_RETRY

Parameters

Returns
The new value of connected (so you should try to connect again if it's not 0).
