43 int usbOpenDevice(usb_dev_handle **device, int vendorID, char *vendorNamePattern, int productID, char *productNamePattern, char *serialNamePattern, FILE *printMatchingDevicesFp, FILE *warningsFp);
44 /* This function iterates over all devices on all USB busses and searches for
45  * a device. Matching is done first by means of Vendor- and Product-ID (passed
46  * in 'vendorID' and 'productID'. An ID of 0 matches any numeric ID (wildcard).
47  * When a device matches by its IDs, matching by names is performed. Name
48  * matching can be done on textual vendor name ('vendorNamePattern'), product
49  * name ('productNamePattern') and serial number ('serialNamePattern'). A
50  * device matches only if all non-null pattern match. If you don't care about
51  * a string, pass NULL for the pattern. Patterns are Unix shell style pattern:
52  * '*' stands for 0 or more characters, '?' for one single character, a list
53  * of characters in square brackets for a single character from the list
54  * (dashes are allowed to specify a range) and if the lis of characters begins
55  * with a caret ('^'), it matches one character which is NOT in the list.
56  * Other parameters to the function: If 'warningsFp' is not NULL, warning
57  * messages are printed to this file descriptor with fprintf(). If
58  * 'printMatchingDevicesFp' is not NULL, no device is opened but matching
59  * devices are printed to the given file descriptor with fprintf().
60  * If a device is opened, the resulting USB handle is stored in '*device'. A
61  * pointer to a "usb_dev_handle *" type variable must be passed here.
62  * Returns: 0 on success, an error code (see defines below) on failure.
63  */
64
