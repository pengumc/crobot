Detailed Description

leg data is unaware of the main body.

The origin point is servo 0, The plane of rotation is the same as the mainbody plate.

int Leg_tryEndpointChange ( leg_t *  leg,
rot_vector_t *  delta 
)

Check if there's a valid solution for a change.

After a successful try you should call Leg_commitEndpointChange to finalize the changes. This is to simplify rollbacks when changing multiple legs.

Parameters:
legThe leg data to use.
deltaThe change in xyz coords of the endpoint. retval 0 Success, Valid coordinates can be set with Leg_commitEndpointChange.
Return values:
-4Error, no valid solution, no changes were made. Please not that the lastResult vector in the solver is not valid now.
-1..3A valid solution was found but one of the servo's couldn't handle the angle. It's number-1 (negative) is returned.
