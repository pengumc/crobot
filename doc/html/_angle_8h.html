int Angle_rangeCheck ( angle_t  test,
angle_t  min,
angle_t  max 
)

Check if an angle is between two other angles.

Angles are in radians.
All values are normalized with Angle_normalize before being used.

Parameters:
testThe angle to test.
minThe minimum angle.
maxThe maximum angle.
Return values:
1Angle lies between min and max.
0Angle lies outside min and max.

Examples: 

  Angle_rangeCheck(0.1, -1.0, 1.0); //will return 1.
  Angle_rangeCheck(3.1, 1.57, -1.57); //checks if 3.1 is on the left half of the unity circle. (it is)
